## Header Slot

Standaard wordt deze template gebruikt om de `functional-header` te bepalen in dit component.

<Source code={header} language="ts" dark={true} />

Als je wijzigingen wil aanbrengen in de functional header, kan je de standaard `vl-functional-header` vervangen door een ander, eventueel aangepaste header element.

Je kan dit bijvoorbeeld vervangen door:
- een [vl-content-header](/docs/components-content-header--content-header-default)
- of een [vl-functional-header](/docs/components-functional-header--functional-header-default) met andere opties dan de standaard `functional-header` van dit component.

In het voorbeeld hieronder kan je zien hoe je voor de `vl-functional-header` bij de teruglink (`data-vl-back`) "Start" als label kan geven in plaats van "Terug".
