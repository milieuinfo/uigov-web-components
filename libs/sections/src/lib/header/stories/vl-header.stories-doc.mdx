# Digitaal Vlaanderen Header

Injecteert de header widget van Digitaal Vlaanderen.

Default wordt de header geïnjecteerd in de body, bij het gebruik van het vl-body element wordt de header geïnjecteerd in de vl-body.  
Indien de header niet getoond wordt op deze pagina, gelieve de pagina te refreshen.

## Voorbeeld

```js
import { VlHeader } from '@domg-wc/sections';
```

```html
<vl-header></vl-header>
```
