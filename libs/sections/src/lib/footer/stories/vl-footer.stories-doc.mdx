# Footer

Injecteert de footer widget van Digitaal Vlaanderen.  
Default wordt de footer geïnjecteerd in de body, bij het gebruik van het vl-body element wordt de footer geïnjecteerd in de vl-body.

## Voorbeeld

```js
import { VlFooter } from '@domg-wc/sections';
```

```html
<vl-footer></vl-footer>
```
