# Map

Gebruik het `map` component om een kaart af te beelden met verschillende lagen en acties.  
**Let op**: De attributen van de `map` worden enkel bij de initiële render geëvalueerd en zijn niet reactief.

## Voorbeeld

```js
import { VlMap } from '@domg-wc/map';
```

```html
<vl-map></vl-map>
```
