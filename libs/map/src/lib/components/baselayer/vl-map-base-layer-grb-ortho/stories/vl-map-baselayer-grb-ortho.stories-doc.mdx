# Map Baselayer GRB Ortho

Gebruik het `map-baselayer-grb-ortho` component om een basis GRB ortho kaartlaag af te beelden.<br/>
Dit component erft over van het [map-baselayer](/docs/map-baselayer--map-baselayer-default) component.

## Voorbeeld

```js
import { VlMapBaseLayerGRBOrtho } from '@domg-wc/map';
```

```html
<vl-map-baselayer-grb-ortho></vl-map-baselayer-grb-ortho>
```
