# Map Baselayer GRB Gray

Gebruik het `map-baselayer-grb-gray` component om een basis GRB kaartlaag met grijstinten af te beelden.<br/>
Dit component erft over van het [map-baselayer](/docs/map-baselayer--map-baselayer-default) component.

## Voorbeeld

```js
import { VlMapBaseLayerGRBGray } from '@domg-wc/map';
```

```html
<vl-map-baselayer-grb-gray></vl-map-baselayer-grb-gray>
```
