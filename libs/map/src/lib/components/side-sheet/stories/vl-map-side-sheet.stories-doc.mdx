# Map Side Sheet

Beeldt een uitklapbaar zijpaneel af over de map. Dit component erft over van het VlSideSheet component.  
Let op: de default positie van de VlMapSideSheet is links, terwijl de default positie van de VlSideSheet rechts is.

## Voorbeeld

```js
import { VlMapSideSheet } from '@domg-wc/map';
```

```html
<vl-map-side-sheet></vl-map-side-sheet>
```
