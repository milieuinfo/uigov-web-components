# Map Overview Map

Gebruik het `map-overview-map` component zodat gebruikers kunnen wisselen tussen de verschillende basis kaartlagen.

## Voorbeeld

```js
import { VlMapOverviewMap } from '@domg-wc/map';
```

```html
<vl-map-overview-map></vl-map-overview-map>
```

<DocsStory id="map-overview-map--map-overview-map-default" />

## Configuratie

Dit component heeft geen configuratie.
