# Map Draw Point Action

Gebruik het `map-draw-point-action` component om een punt te tekenen op een [map-features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default).<br/>
Dit component erft over van de `VlMapDrawAction` klasse, die op zijn beurt overerft van de `VlMapLayerAction` klasse, die op zijn beurt overerft van de `VlMapAction` klasse.

## Voorbeeld

```js
import { VlMapDrawPointAction } from '@domg-wc/map';
```

```html
<vl-map-draw-point-action></vl-map-draw-point-action>
```
