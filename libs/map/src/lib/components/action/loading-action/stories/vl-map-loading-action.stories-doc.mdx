
# Loading Action

Vangt op wanneer de `VlMap` aan het laden is (via de loadstart en loadend events van OL). Zal een 'loading' bar bovenaan de map zetten.

## Voorbeeld

```js
import { VlMapLoadingAction } from '@domg-lib/components'
```

```html
        <vl-map>
            <vl-map-baselayer-grb-gray></vl-map-baselayer-grb-gray>
            <vl-map-loading-action></vl-map-loading-action>
        </vl-map>
```
