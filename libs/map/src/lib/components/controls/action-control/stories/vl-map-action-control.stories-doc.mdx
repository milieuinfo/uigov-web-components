# Map Action Control

Gebruik het `map-action-control` component om een map-actie aan of uit te zetten door behulp van een [toggle-button](/docs/components-toggle-button--toggle-button-default).<br/>
Een `map-action-control` linken aan een map-actie gebeurt op de volgende manier:

-   Plaats op de `map-action-control` het attribuut `data-vl-action-id` met als waarde bv. 'draw-polygon-action'
-   Plaats op de map-actie het attribuut `id` en geef hier dezelfde waarde mee als in de vorige stap

## Voorbeeld

```js
import { VlMapActionControl } from '@domg-wc/map';
```

```html
<vl-map-action-control></vl-map-action-control>
```
