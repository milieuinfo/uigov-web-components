# Map Baselayer GRB

<VluxMetaData id="map-baselayer-baselayer-grb" />

Gebruik het `map-baselayer-grb` component om een basis GRB kaartlaag af te beelden.<br/>
Dit component erft over van het [map-baselayer](/docs/map-baselayer--map-baselayer-default) component.


## Voorbeeld

```js

```

