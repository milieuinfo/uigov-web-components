# Map Search

<CustomDocMetaData id="map-search" />

Gebruik de `map-search` component om een zoekbalk toe te voegen aan de map waarmee gebruikers kunnen zoeken op adres.<br/>
Deze component maakt gebruik van de [select-location](/docs/map-select-location--select-location-default) component en
van de [search](/docs/components-search--search-default) component.


## Voorbeeld

```js
import { VlMapSearch } from '@domg-wc/map';
```

```html
<vl-map-search></vl-map-search>
```
