# Map Search

Gebruik het `map-search` component om een zoekbalk toe te voegen aan de map waarmee gebruikers kunnen zoeken op adres.<br/>
Dit component maakt gebruik van het [select-location](/docs/map-select-location--select-location-default) component en van het [search](/docs/components-search--search-default) component.

## Voorbeeld

```js
import { VlMapSearch } from '@domg-wc/map';
```

```html
<vl-map-search></vl-map-search>
```
