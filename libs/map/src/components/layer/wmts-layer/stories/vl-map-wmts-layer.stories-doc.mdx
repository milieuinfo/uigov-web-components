# Map WMTS Layer

Gebruik het `map-wmts-layer` component om een WMTS kaartlaag af te beelden.<br/>
Dit component erft over van de `VlMapLayer` klasse.

## Voorbeeld

```js
import { VlMapWmtsLayer } from '@domg-wc/map';
```

```html
<vl-map-wmts-layer></vl-map-wmts-layer>
```
