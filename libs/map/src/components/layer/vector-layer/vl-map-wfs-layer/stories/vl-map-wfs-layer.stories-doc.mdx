# Map WFS Layer

Gebruik het `map-wfs-layer` component om een WFS kaartlaag af te beelden.<br/>
Dit component erft over van de `VlMapVectorLayer` klasse, die op zijn beurt overerft van de `VlMapLayer` klasse.

## Voorbeeld

```js
import { VlMapWfsLayer } from '@domg-wc/map';
```

```html
<vl-map-wfs-layer></vl-map-wfs-layer>
```
