# Map Loading Indicator

<VluxMetaData id="map-loading-indicator" />

Gebruik de `map-loading-indicator` component om op te vangen wanneer de [vl-map](/docs/map-map--map-default) aan het
laden is.<br/>
Toont een 'loading' bar bovenaan de map.


## Voorbeeld

```js
import { VlMapLoadingIndicator } from '@domg-lib/map';
```

```html
<vl-map-loading-indicator></vl-map-loading-indicator>
```
