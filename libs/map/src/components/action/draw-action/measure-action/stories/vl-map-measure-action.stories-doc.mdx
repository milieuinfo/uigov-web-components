# Map Measure Action

<CustomDocMetaData id="map-action-draw-action-measure-action" />

Gebruik het `map-measure-action` component om de afstand tussen 2 punten te meten op een
[map-features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default).<br/>
Dit component erft over van de `VlMapDrawAction` klasse, die op zijn beurt overerft van de `VlMapLayerAction` klasse,
die op zijn beurt overerft van de `VlMapAction` klasse.


## Voorbeeld

```js
import { VlMapMeasureAction } from '@domg-wc/map';
```

```html
<vl-map-measure-action></vl-map-measure-action>
```
