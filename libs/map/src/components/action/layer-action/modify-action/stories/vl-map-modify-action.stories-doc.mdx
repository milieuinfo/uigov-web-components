# Map Modify Action

<VluxMetaData id="map-action-layer-action-modify-action" />

Gebruik het `map-modify-action` component om een feature op een [map-features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default) aan te passen.<br/>
Dit component erft over van de `VlMapLayerAction` klasse, die op zijn beurt overerft van de `VlMapAction` klasse.


## Voorbeeld

```js
import { VlMapModifyAction } from '@domg-wc/map';
```

```html
<vl-map-modify-action></vl-map-modify-action>
```
