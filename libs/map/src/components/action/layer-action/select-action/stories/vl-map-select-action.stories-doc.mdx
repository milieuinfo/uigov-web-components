# Map Select Action

<VluxMetaData id="map-action-layer-action-select-action" />

Gebruik het `map-select-action` component om een feature op een [map-features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default) te selecteren.<br/>
Dit component erft over van de `VlMapLayerAction` klasse, die op zijn beurt overerft van de `VlMapAction` klasse.


## Voorbeeld

```js
import { VlMapSelectAction } from '@domg-wc/map';
```

```html
<vl-map-select-action></vl-map-select-action>
```

### Clustering

Bij clustering kan best een [map-layer-circle-style](/docs/map-layer-style-layer-circle-style--map-layer-circle-style-default)
component gebruikt worden binnen de `vl-map-select-action` tag.<br/>

Dit zorgt ervoor dat de grootte van de select-actie zich aanpast aan het aantal geclusterde features, en dat de stijl-tekst getoond wordt.
