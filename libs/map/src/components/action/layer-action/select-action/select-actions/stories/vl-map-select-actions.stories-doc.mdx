# Map Select Actions

<CustomDocMetaData id="map-action-layer-action-select-action-select-actions" />

Gebruik het `map-select-actions` component om features op meerdere
[map-features-layers](/docs/map-layer-vector-layer-features-layer--map-features-layer-default) te selecteren.<br/>
Deze component erft over van de
[map-select-action](/docs/map-action-layer-action-select-action--map-select-action-default) component.


## Voorbeeld

```js
import { VlMapSelectActions } from '@domg-wc/map';
```

```html
<vl-map-select-actions></vl-map-select-actions>
```

### Clustering

Bij clustering kan best een [map-layer-circle-style](/docs/map-layer-style-layer-circle-style--map-layer-circle-style-default) component gebruikt worden binnen de `vl-map-select-actions` tag.<br/>
Dit zorgt ervoor dat de grootte van de select-actie zich aanpast aan het aantal geclusterde features, en dat de stijl-tekst getoond wordt.
