# Map Click Action

Gebruik het `map-click-action` component om een klikactie op de [map](/docs/map-map--map-default) op te vangen.<br/>
Bij het klikken op de [map](/docs/map-map--map-default) wordt er een `vl-map-click-action-pindrop` op de kaart geplaatst.

## Voorbeeld

```js
import { VlMapClickAction } from '@domg-wc/map';
```

```html
<vl-map-click-action></vl-map-click-action>
```
