# Map Current Location

Gebruik het `map-current-location` component om een huidige locatie knop af te beelden op de map.<br/>
Na het drukken op de knop wordt de huidige locatie van de gebruiker centraal afgebeeld.


## Voorbeeld

```js
import { VlMapCurrentLocation } from '@domg-wc/map';
```

```html
<vl-map-current-location></vl-map-current-location>
```
