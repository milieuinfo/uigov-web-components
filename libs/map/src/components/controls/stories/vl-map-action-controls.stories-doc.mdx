# Map Action Controls

<CustomDocMetaData id="map-controls-action-controls" />

Gebruik de `map-action-controls` component als wrapper rond
[map-action-control](/docs/map-controls-action-control--map-action-control-default) componenten.<br/>
De `map-action-controls` zorgt ervoor dat de map-acties in de rechterbovenhoek van de `map` afgebeeld worden.<br/>
Verder zorgt deze ervoor dat er altijd maar 1 actieve `map-action-control` is.


## Voorbeeld

```js
import { VlMapActionControls } from '@domg-wc/map';
```

```html
<vl-map-action-controls></vl-map-action-controls>
```

<Canvas of={VlMapActionControlsStories.MapActionControlsDefault} />


## Configuratie

Deze component heeft geen configuratie.
