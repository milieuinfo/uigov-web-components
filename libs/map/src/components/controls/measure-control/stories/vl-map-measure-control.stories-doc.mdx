# Map Measure Control

<CustomDocMetaData id="map-controls-measure-control" />

Gebruik de `map-measure-control` component om de meet-actie aan of uit te zetten door behulp van een
[toggle-button](/docs/components-toggle-button--toggle-button-default).


## Voorbeeld

```js
import { VlMapMeasureControl } from '@domg-wc/map';
```

```html
<vl-map-measure-control></vl-map-measure-control>
```

<Canvas of={VlMapMeasureControlStories.MapMeasureControlDefault} />


## Configuratie

Deze component heeft geen configuratie.
