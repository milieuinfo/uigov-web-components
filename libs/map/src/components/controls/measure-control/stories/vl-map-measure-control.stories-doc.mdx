# Map Measure Control

Gebruik het `map-measure-control` component om de meet-actie aan of uit te zetten door behulp van een [toggle-button](/docs/components-toggle-button--toggle-button-default).

## Voorbeeld

```js
import { VlMapMeasureControl } from '@domg-wc/map';
```

```html
<vl-map-measure-control></vl-map-measure-control>
```

<DocsStory id="map-controls-measure-control--map-measure-control-default" />

## Configuratie

Dit component heeft geen configuratie.
