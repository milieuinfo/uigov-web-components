# Map Overview Map

<VluxMetaData id="map-overview-map" />

Gebruik de `map-overview-map` component zodat gebruikers kunnen wisselen tussen de verschillende basis kaartlagen.


## Voorbeeld

```js
import { VlMapOverviewMap } from '@domg-wc/map';
```

```html
<vl-map-overview-map></vl-map-overview-map>
```

<Canvas of={VlMapOverviewMapStories.MapOverviewMapDefault}/>


## Configuratie

Deze component heeft geen configuratie.
