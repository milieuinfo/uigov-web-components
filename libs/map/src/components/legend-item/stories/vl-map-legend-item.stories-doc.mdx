# Map Legend Item

<VluxMetaData id="map-legend-item" />

Gebruik de `map-legend-item` component om een custom legende af te beelden op de kaart.

Indien geen icon- en labelslots meegegeven zijn zal het `map-legend-item` de default style items van de bijhorende laag tonen.


## Voorbeeld

```js
import { VlMapLegendItem } from '@domg-wc/map';
```

```html
<vl-map-legend-item></vl-map-legend-item>
```
