# Select Location

Gebruik het `select-location` component om een zoekbalk af te beelden waarmee er een adres geselecteerd kan worden.<br/>
Dit component erft over van het [select](/docs/elements-select--select-default) component.

## Voorbeeld

```js
import { VlSelectLocation } from '@domg-wc/map';
```

```html
<select is="vl-select-location"></select>
```
