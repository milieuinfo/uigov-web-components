# Map Layer Style

Gebruik de `map-layer-style` component om een
[features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default) of een
[wfs-layer](/docs/map-layer-vector-layer-wfs-layer--map-wfs-layer-default) te stylen.<br/>
**Let op**: Bij gebruik van de [map-legend](/docs/map-legend--map-legend-features-layer-multiple-styles) worden de
attributen niet reactief toegepast op de legende.


## Voorbeeld

```js
import { VlMapLayerStyle } from '@domg-wc/map';
```

```html
<vl-map-layer-style></vl-map-layer-style>
```
