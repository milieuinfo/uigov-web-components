# Visual

Gebruik de <code>vl-qlik-infoblock</code> om een infoblock te maken met icoon, titel, eventuele tekst en verschillende visualisaties.s

## Voorbeeld

```js
import '@domg-wc/qlik/infoblock';
import viz_without_views from './visuals.json' assert {type: 'json'}

const connection = new Qlik("omgevingsloketrapport.omgeving.vlaanderen.be", "594c04b6-f319-4cb9-962c-24b0f7aa6f5e");
await connection.init();
this.connection = connection;
```

```html
<vl-template>
        <vl-qlik-infoblock title="Marie Jeanne" icon="airplane" .connection="${connection}" .visuals="${viz_without_views}">
            <span>Do something</span>
        </vl-qlik-infoblock>
</vl-template>
```
