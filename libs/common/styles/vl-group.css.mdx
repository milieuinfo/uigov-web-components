# vl-group

## wat

Te bespreken, conform form-group / input-group / link-list / accordion-list / ?

* action-group -> 'css'-component: 'vl-group' [UIG-2907]
* form-group -> 'css'-component: 'vl-group' [geen expliciet ticket]
* input-group -> 'css'-component: 'vl-group' [UIG-2922]
* accordion-list -> 'css'-component: 'vl-group' [geen expliciet ticket]
* link-list -> 'css'-component: 'vl-group' [UIG-2925]
* icon-wrapper -> 'css'-component: 'vl-group' [geen expliciet ticket]


## action-group [UIG-2907]
    vl-align
    vl-space-between
    vl-bordered
    vl-collapse-l
    vl-collapse-m
    vl-collapse-s
    vl-collapse-xs

-> al deze styles moeten mogelijk zijn in vl-group


## form-group [geen expliciet ticket]

Met vl-form-group wordt styling toegevoegd om makkelijker te onderscheiden tussen verschillende form controls.

    vl-escape-field-names
    vl-native-validation
    vl-validate

-> wordt geschrapt, wel een manier voorzien om de group een achtergrond kleur te geven


## input-group [UIG-2922]

-> te bekijken hoe dit moet / kan met vl-group


## accordion-list [geen expliciet ticket]

Een lijst component die kan gebruikt worden als je meerdere accordions wilt afbeelden. Voor meer informatie over accordions zie de accordion component.

    vl-bordered

-> bordered is een horizantale lijn tussen 2 items
-> direction voorzien, afhankelijk van de gekozen richting zal de lijn omgekeerd staan (horizontaal / verticaal)


## link-list [UIG-2925]
    vl-small
    vl-inline
    vl-bordered

-> inline is mogelijks direction=horizontal (of row)
-> small verkleint de font-size van de group


## icon-wrapper [geen expliciet ticket]

Gebruik de icon-wrapper als parent element om al de iconen verticaal te aligneren.

-> is direction=vertical (of column)
