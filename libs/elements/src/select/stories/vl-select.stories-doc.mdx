# Select

‚ùóDEPRECATED: Gebruik de [vl-select-next](/docs/form-next-select--documentatie), in v2.0.0 verdwijnt deze component.

Gebruik de `select` component om een gebruiker 1 keuze te laten maken uit een voorgedefinieerde lijst.<br/>
Het is aangeraden deze component enkel te gebruiken bij 5 of meer opties. Bij minder opties heeft de radio component
de voorkeur.


## Voorbeeld

```js
import { VlSelect } from '@domg-wc/elements';
```

```html
<select is="vl-select"></select>
```

## Varianten

### Uitgebreid

Je kan de uitgebreide select-functionaliteit inclusief zoeken activeren door het `data-vl-select` attribuut mee te
geven.<br/>
In deze modus wordt er achterliggend gebruik gemaakt van [Choices.js](https://github.com/Choices-js/Choices).
