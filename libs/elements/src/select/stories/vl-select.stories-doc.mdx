# Select

Gebruik het `select` component om gebruikers toe te laten een selectie te maken uit een lijst met voorgedefinieerde opties.<br/>
Het is aangeraden om enkel deze component te gebruiken als er 5 of meer opties zijn. Bij minder opties, kan er gebruik gemaakt worden van de radio component.

## Voorbeeld

```js
import { VlSelect } from '@domg-wc/elements';
```

```html
<vl-select></vl-select>
```

## Varianten

### Uitgebreid

Je kan de uitgebreide select-functionaliteit inclusief zoeken activeren door het `data-vl-select` attribuut mee te geven.<br/>
In deze modus wordt er achterliggend gebruik gemaakt van [Choices.js](https://github.com/Choices-js/Choices).
