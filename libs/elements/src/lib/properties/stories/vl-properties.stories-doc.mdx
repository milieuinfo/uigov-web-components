# Properties

Vormt de container van een lijst van kenmerken van een onderwerp.  
Deze component wordt meestal gebruikt om informatie te tonen dat ingevuld werd in een formulier.

## Voorbeeld

```js
import { VlPropertiesComponent } from '@domg-wc/elements';
```

