# Input Field

Gebruik het `input-field`-component om tekst van de gebruiker op te vangen.

## Voorbeeld

```js
import { VlUploadComponent } from '@domg-wc/components';
```

```html
 <input is="vl-input-field" />
```

## Form Validatie

De `input-field`-component ook `vl-form-validation` element, die achterliggend op zijn beurt gebruik maakt van `validate.js` 0.12 ([http://validatejs.org/](http://validatejs.org/)).

[Link naar `vl-form-validation`-component](/docs/elements-form--form-validation) documentatie in Storybook.

## Varianten

### Binnen een form

Gelijkaardig aan de andere controls die overerven van de [`vl-form-validation`](/docs/elements-form--form-validation) kan je de `vl-form-validation-message` gebruiken om een foutboodschap te tonen.
