# Side Navigation

Een compact navigatie-element dat je aan een pagina kan toevoegen. Het vat de inhoud van lange pagina's samen, leidt de gebruiker door de pagina inhoud en kan ook naar externe pagina's verwijzen.

Het zij-menu wordt opgebouwd uit volgende elementen:

-   VlSideNavigation
-   VlSideNavigationTitle
-   VlSideNavigationContent
-   VlSideNavigationGroup
-   VlSideNavigationItem
-   VlSideNavigationToggle
-   VlSideNavigationReference

Zie de [code voorbeelden](#code-voorbeelden) onderaan.

```js
import { VlSideNavigation } from '@domg-wc/elements';
```

```html
<nav is="vl-side-navigation"></nav>
```

## Responsive variant

Op mobiel zal de `vl-side-navigation` een knop bovenaan tonen die de gebruiker de mogelijkheid geeft het navigatie-menu te zien.
Als de gebruiker scrollt, zal deze knop verdwijnen en komt na een korte delay een sticky knop met dezelfde functionaliteit. Deze knop blijft zichtbaar zolang de gebruiker scrollt in een gebied gerelateerd tot de `vl-side-navigation-reference`.

## Parent en child links

Het is mogelijk om parent en child links te gebruiken (zie content 1 en content 2).
Dit gebeurt op de volgende manier (we gebruiken content 1 als voorbeeld):

1. Plaats op het parent VlSideNavigationItem element het attribuut `data-vl-parent="content-1"`
1. Plaats op het VlSideNavigationToggle element het attribuut `data-vl-child="content-1"`
1. Plaats op de child links het attribuut `data-vl-parent="content-1"`

Zie de [code voorbeelden](#code-voorbeelden) onderaan.
