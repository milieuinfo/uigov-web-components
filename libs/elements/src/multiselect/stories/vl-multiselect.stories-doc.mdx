# MultiSelect

<VluxMetaData id="elements-multiselect" />

Gebruik de `multiselect` component om een gebruiker 1 of meerdere keuzes te laten maken uit een voorgedefinieerde lijst.


## Voorbeeld

```js
import { VlMultiSelect } from '@domg-wc/elements';
```

```html
<select is="vl-multiselect"></select>
```
