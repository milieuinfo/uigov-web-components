# Icon Wrapper

<CustomDocMetaData id="elements-icon-icon-wrapper" />

Gebruik de `icon-wrapper` als parent element om al de iconen verticaal te aligneren.


## Voorbeeld

```js
import { VlIconWrapperElement } from '@domg-wc/elements';
```

```html
<p is="vl-icon-wrapper">
    <span is="vl-icon" data-vl-icon="calendar"></span>
    <span is="vl-icon" data-vl-icon="calendar"></span>
    <span is="vl-icon" data-vl-icon="calendar"></span>
</p>
```
