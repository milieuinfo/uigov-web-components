# Input Field Masked - next

Gebruik de `input-field-masked-next` component om een input veld met een mask toe te voegen aan een pagina.<br/>
Zie het [form demo](/docs/applicatief-voorbeelden-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component erft over van de [input-field-next](/docs/form-next-input-field--documentatie) component.

## Voorbeeld

```js
import { VlInputFieldMaskedComponent } from '@domg-wc/form/next/input-field-masked';
```

```html
<vl-input-field-masked-next></vl-input-field-masked-next>
```

## Publieke methodes

### getRawValue(): string

Geeft de raw value van de input terug (zonder mask).<br/>
Bij het `price` mask wordt de prefix weggelaten.

## Validatie

- Bij het gebruik in een form wordt er automatisch mask validatie toegevoegd aan het input veld.
- De `patternMismatch` ValidityState key wordt gebruikt voor de mask validatie.
- Het 'pattern' attribuut wordt uitgeschakeld omdat het mask deze werking overneemt, gelieve dit attribuut niet te gebruiken.
- Het is mogelijk om de mask validatie te combineren met andere validaties (bv. `required`).
