# Textarea - next

Gebruik de `textarea-next` component om een textarea veld toe te voegen aan een pagina.<br/>
Zie het [form demo](/docs/ontwerp-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component is de volgende versie van de [vl-textarea](/docs/components-textarea--documentatie) component, we raden aan op termijn deze versie te gebruiken.<br />
In tegenstelling tot de vorige [vl-textarea](/docs/components-textarea--documentatie) bevat dit component geen rich-text modus, hier wordt op termijn een ander component voor voorzien.

## Voorbeeld

```js
import { VlTextareaComponent } from '@domg-wc/form/next/textarea';
```

```html
<vl-textarea-next></vl-textarea-next>
```
