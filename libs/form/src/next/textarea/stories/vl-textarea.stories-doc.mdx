# Textarea

Gebruik de `textarea-next` component om een textarea veld toe te voegen aan een pagina.<br/>
Deze component is de opvolger van de [vl-textarea](/docs/components-textarea--documentatie) component, we raden aan
deze nieuwe versie te gebruiken.<br />
In tegenstelling tot de vorige [vl-textarea](/docs/components-textarea--documentatie) bevat deze component geen
rich-text mode, op termijn wordt hiervoor een andere component voorzien.

## Voorbeeld

```js
import { VlTextareaComponent } from '@domg-wc/form/next/textarea';
```

```html
<vl-textarea-next></vl-textarea-next>
```
