# Textarea - next

<VluxMetaData id="form-next-textarea" />

Gebruik de `textarea-next` component om een textarea veld toe te voegen aan een pagina.<br/>
Zie het [form demo](/docs/ontwerp-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Deze component bevat geen rich-text modus: gebruik daarvoor
[vl-textarea-rich-next](/docs/form-next--text-area-rich--documentatie).


## Voorbeeld

```js
import { VlTextareaComponent } from '@domg-wc/form/next/textarea';
```

```html
<vl-textarea-next></vl-textarea-next>
```
