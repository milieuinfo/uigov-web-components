# Input Field - next

Gebruik de `input-field-next` component om een input veld toe te voegen aan een pagina.<br/>
Zie het [form demo](/docs/applicatief-voorbeelden-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component is de volgende versie van de [vl-input-field](/docs/elements-input-field--documentatie) component, we raden aan deze versie te gebruiken.

## Voorbeeld

```js
import { VlInputFieldComponent } from '@domg-wc/form/next/input-field';
```

```html
<vl-input-field-next></vl-input-field-next>
```
