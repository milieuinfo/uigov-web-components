# Form Label - next

<VluxMetaData id="form-next-form-label" />

Gebruik de `form-label-next` component om een form label af te beelden.<br/>
Zie het [form demo](/docs/ontwerp-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component is de volgende versie van de [vl-form-label](/docs/elements-form-message-form-label--documentatie) component, we raden aan deze versie te gebruiken.


## Voorbeeld

```js
import { VlFormLabelComponent } from '@domg-wc/form/next/form-label';
```

```html
<vl-form-label-next></vl-form-label-next>
```
