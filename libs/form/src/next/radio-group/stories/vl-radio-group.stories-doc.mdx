# Radio Group - next

<VluxMetaData id="form-next-radio-group" />

Gebruik de `radio-group-next` component om de gebruiker de mogelijkheid te geven om 1 keuze te selecteren in een lijst.<br/>
Wanneer mogelijk, selecteer geen `radio` van vooraf zodat de gebruiker een bewuste keuze kan maken.<br/>
De `radio-next` component is een onderdeel van de `radio-group-next` component.<br/>
Zie het [form demo](/docs/ontwerp-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component is de volgende versie van het [vl-radio](/docs/components-radio--documentatie) component, we raden aan op termijn deze versie te gebruiken.


## Voorbeeld

```js
import { VlRadioComponent, VlRadioGroupComponent } from '@domg-wc/form/next/radio-group';
```

```html
<vl-radio-group-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
</vl-radio-group-next>
```

### Required

De `radio-group-next` component kan ingesteld worden als `required`. Dit betekent dat er minstens 1 `vl-radio-next` geselecteerd moet worden.
