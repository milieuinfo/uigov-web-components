# Radio

Gebruik de `radio-group-next` component om 1 keuze te selecteren in een lijst. Je zet bij voorkeur geen default,
zodat de gebruiker een bewuste keuze dient te maken.

De `radio-next` component is een onderdeel van de `radio-group-next` component.

Deze component is de opvolger van de [vl-radio](?path=/story/components-next-form-radio--radio-default)
component, we raden aan deze nieuwe versie te gebruiken.


## Voorbeeld

```js
import { VlRadioComponent, VlRadioGroupComponent } from '@domg-wc/form/next/radio';
```

```html
<vl-radio-group-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
</vl-radio-group-next>
```
