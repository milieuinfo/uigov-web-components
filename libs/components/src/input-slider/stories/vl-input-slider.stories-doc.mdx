# Input Slider

<VluxMetaData id="components-input-slider" />

Gebruik de `input-slider` om een waarde te selecteren tussen een minimum en maximum waarde.
**Let op**: deze component is niet dezelfde als de `vl-ui-range` component van Digitaal Vlaanderen.


## Voorbeeld

```js
import { VlInputSliderComponent } from '@domg-wc/components';
```

```html
<vl-input-slider></vl-input-slider>
```

## Referenties

Dit is een nieuwe component binnen onze bibliotheek die gebaseerd is op de standaard [HTML5 input range slider](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/range).

De stijl van deze component is gebaseerd op de `vl-ui-range` [component van Digitaal Vlaanderen](https://overheid.vlaanderen.be/webuniversum/v3/documentation/forms/vl-ui-range).
