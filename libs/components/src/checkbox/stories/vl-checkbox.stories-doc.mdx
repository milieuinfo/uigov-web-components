# Checkbox

Gebruik een `checkbox` om de gebruiker de mogelijkheid te geven 1 of meerdere items te selecteren uit een lijst.


Deze [vl-checkbox](?path=/story/components-checkbox--checkbox-default) versie is behouden voor backwards-compatibility, in v2.0.0 gaat dit verdwijnen.<br/>
Dit is de volgende versie van het [vl-checkbox](?path=/docs/components-next-form-checkbox--checkbox-default) component (`vl-checkbox-next`), we raden aan deze versie te gebruiken.<br />


## Voorbeeld

```js
import { VlCheckboxComponent } from '@domg-wc/components';
```

```html
<vl-checkbox></vl-checkbox>
```
