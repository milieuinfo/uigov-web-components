# Document

Gebruik de `document` component om een link naar een bestand toe te voegen dat de gebruiker in de browser kan bekijken
of downloaden.


## Voorbeeld

```js
import { VlDocumentComponent } from '@domg-wc/components';
```

```html
<vl-document></vl-document>
```
