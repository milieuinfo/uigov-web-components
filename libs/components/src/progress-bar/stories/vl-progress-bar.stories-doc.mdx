# Progress Bar

Gebruik een progress-bar om de vooruitgang te tonen van een proces dat uit verschillende stappen bestaat.

Standaard worden stappen alleen getoond als je er met de muis overheen gaat. Dit kan worden geconfigureerd met `data-vl-show-steps`.

## Voorbeeld

```js
import { VlProgressBarComponent } from '@domg-wc/components';
```

```html
<vl-progress-bar></vl-progress-bar>
```
