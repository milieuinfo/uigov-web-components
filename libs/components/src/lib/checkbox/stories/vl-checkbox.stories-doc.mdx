# Checkbox

Gebruik een `checkbox` om de gebruiker de mogelijkheid te geven om 1 of meerdere dingen te selecteren in een lijst.

## Voorbeeld

```js
import { VlCheckboxComponent } from '@domg-wc/components';
```

```html
<vl-checkbox></vl-checkbox>
```
