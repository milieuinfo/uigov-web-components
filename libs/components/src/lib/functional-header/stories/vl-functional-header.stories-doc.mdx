### Met tabs

Je kan de `vl-tabs` gebruiken in het sub-header slot om tabs af te beelden binnen de functional header.  
Het klikken op een tab vuurt enkel een event af en past de actieve tab niet aan, hiervoor moet je het attribuut `data-vl-active-tab` aanpassen.

Voor meer informatie zie het [tabs](/story/components-tabs--tabs-default) component.
