# Functional Header

Gebruik het `functional-header` component om bovenaan de pagina generieke informatie te tonen zoals bijvoorbeeld titel en acties.

## Voorbeeld

```js
import { VlFunctionalHeaderComponent } from '@domg-lib/components';
```

```html
<vl-functional-header></vl-functional-header>
```

### Met tabs

Gebruik het [vl-tabs](/docs/components-tabs--tabs-default) component in het sub-header slot om tabs af te beelden binnen de `functional-header`.<br/>
Het klikken op een tab vuurt enkel een event af en past de actieve tab niet aan, hiervoor moet je het attribuut `data-vl-active-tab` aanpassen.<br/>
