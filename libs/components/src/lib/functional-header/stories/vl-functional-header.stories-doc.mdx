# Functional Header

Gebruik het `functional-header` component om bovenaan de pagina generieke informatie te tonen zoals bijvoorbeeld titel en acties.

## Voorbeeld

```js
import { VlFunctionalHeaderComponent } from '@domg-lib/components';
```

```html
<vl-functional-header></vl-functional-header>
```
