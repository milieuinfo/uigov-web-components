# Progress Bar

Gebruik een progress-bar om de vooruitgang te tonen van een proces dat uit verschillende stappen bestaat.


## Voorbeeld

```js
import { VlProgressBarComponent } from '@domg-lib/components'
```

```html
    <vl-progress-bar data-vl-active-step="1"></vl-progress-bar>
```
