# Accordion

Gebruik het `accordion` component om informatie te tonen of te verbergen aan de hand van een toggle.  
Voor meer informatie over het afbeelden van meerdere accordions zie het [accordion-list](/story/components-accordion-list--accordion-list-default) component.

## Voorbeeld

```js
import { VlAccordionComponent } from '@domg-wc/components';
```

```html
<vl-accordion></vl-accordion>
```
