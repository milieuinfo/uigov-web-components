# Content Header

Gebruik het `content-header` component bovenaan informerende pagina's. Het biedt de gebruikers alle informatie over de website die ze bezoeken.

## Voorbeeld

```js
import { VlContentHeaderComponent } from '@domg-lib/components';
```

```html
<vl-content-header></vl-content-header>
```
