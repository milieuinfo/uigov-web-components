# Tabs

Gebruik tabs om gerelateerde informatie op te splitsen in kleinere stukken content.  
Op mobiel wordt de tab navigatie omgevormd tot een uitklapbaar menu.

## Voorbeeld

```js
import { VlTabsComponent } from '@domg-wc/components';
```

```html
<vl-tabs></vl-tabs>
```
