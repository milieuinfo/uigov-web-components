# Input Slider

Gebruik een `input-slider` om een waarde te selecteren tussen een minimum en maximum waarde aan de hand van een slider.  
**Let op**: dit component is niet hetzelfde component als het `vl-ui-range` component van Digitaal Vlaanderen.

## Voorbeeld

```js
import { VlInputSliderComponent } from '@domg-wc/components';
```

```html
<vl-input-slider></vl-input-slider>
```
