# Wizard

Gebruik een wizard om een gebruiker door een meerstapsproces te begeleiden.
Een wizard maakt het mogelijk om een ingewikkeld proces op te delen in overzichtelijke, kleine stappen.
Ook biedt een wizard de mogelijkheid om de opties in een bepaalde stap aan te passen op basis van de keuzes die de gebruiker in de voorgaande stap heeft gemaakt.
Een wizard leid een gebruiker door een proces met meerdere stappen.
Standaard worden stappen altijd getoond op de `vl-wizard-*`. Dit is niet configureerbaar.

## Voorbeeld

```js
import { VlWizard } from '@domg-wc/components';
```

```html
<vl-wizard>
    <vl-wizard-pane data-vl-name="Step 1"><p>Pane content 1</p></vl-wizard-pane>
    <vl-wizard-pane data-vl-name="Step 2"><p>Pane content 2</p></vl-wizard-pane>
</vl-wizard>
```
