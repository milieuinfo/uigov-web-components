# Wizard Pane

Gebruik de wizard-pane component om een stap in de wizard af te beelden.
Te gebruiken in combinatie met de [wizard](/story/components-wizard-wizard--documentatie) component


## Voorbeeld

```js
import { VlWizardPane } from '@domg-wc/components';
```

```html
<vl-wizard>
    <vl-wizard-pane data-vl-name="Step 1"><p>Pane content</p></vl-wizard-pane>
</vl-wizard>
```
