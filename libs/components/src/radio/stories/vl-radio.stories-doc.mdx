# Radio

De `radio` laat de gebruiker toe om een enkele optie te selecteren uit een lijst. Gebruik de `radio` in formulieren. Vermijd een voorgedefinieerde keuze vast te leggen om de gebruiker een bewuste keuze te laten maken.

## Voorbeeld

```js
import { VlRadio } from '@domg-wc/components';
```

```html
<vl-radio></vl-radio>
```
