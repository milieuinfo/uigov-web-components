# Radio

De `radio` laat de gebruiker toe om een enkele optie te selecteren uit een lijst. Gebruik de `radio` in formulieren.<br/>
Vermijd een voorgedefinieerde keuze vast te leggen zodat de gebruiker een bewuste keuze dient te maken.
