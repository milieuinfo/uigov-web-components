# Button - next

Gebruik de `button-next` component om een button af te beelden op een pagina.

Dit component is de volgende versie van de [vl-button](/docs/elements-button-button--documentatie) component.

## Voorbeeld

```js
import { VlButtonComponent } from '@domg-wc/components/next/button';
```

```html
<vl-button-next></vl-button-next>
```

### Loading

Gebruik een loading button om aan te tonen dat een actie enige tijd in beslag neemt.<br/>
Gedurende het uitvoeren van de actie kan je de button als een loading button afbeelden.<br/>
We raden aan een loading button enkel te gebruiken voor primary buttons.<br/>
Indien je wil dat de loading button ook disabled is, kan je het `disabled` attribuut in combinatie met het `loading` attribuut gebruiken.

### Toggle

Gebruik een toggle button om aan te duiden dat een actie aan of uit staat.<br/>
De button wordt aan- of uitgezet door er op te klikken.<br/>
Gebruik het `on` attribuut om de button programmatorisch aan- of uit te zetten.<br/>
Gebruik het `controlled` attribuut als je wil dat de button zich niet zelf aan- of uitzet, gebruik dan het `on` attribuut om de button aan- of uit te zetten.
