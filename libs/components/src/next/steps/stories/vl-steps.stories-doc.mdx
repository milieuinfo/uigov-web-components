# Steps Next

Gebruik het `steps-next` component om een verticale lijst van stappen af te beelden om de gebruiker door een procedure te begeleiden.<br/>
Dit component is de volgende versie van het [vl-steps](?path=/docs/components-steps--steps-default) component, we raden aan deze versie te gebruiken.<br />
De oude [vl-steps](?path=/docs/components-steps--steps-default) versie is behouden voor backwards-compatibility, in v2.0.0 gaat dit verdwijnen.<br/><br/>

## Voorbeeld

```js
import { VlStepsComponent } from '@domg-wc/components/next/steps';
```

```html
<vl-steps-next></vl-steps-next>
```
