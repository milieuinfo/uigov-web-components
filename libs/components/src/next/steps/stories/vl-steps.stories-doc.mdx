# Steps Next

Gebruik de `steps-next` component om een verticale lijst van stappen af te beelden om de gebruiker door een procedure
te begeleiden.<br/>
Deze component is de nieuwste versie van de [vl-steps](?path=/docs/components-steps--steps-default) component, we raden
aan deze versie te gebruiken.<br />
De oude [vl-steps](?path=/docs/components-steps--steps-default) versie blijft behouden voor backwards-compatibility,
in v2.0.0 zal deze verdwijnen.<br/><br/>


## Voorbeeld

```js
import { VlStepsComponent } from '@domg-wc/components/next/steps';
```

```html
<vl-steps-next></vl-steps-next>
```
