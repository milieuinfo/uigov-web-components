# Steps - next

Gebruik de `steps-next` component om een verticale lijst van stappen af te beelden om de gebruiker door een procedure te begeleiden.<br/>

Deze component is de nieuwe versie van de [vl-steps](/docs/components-steps--documentatie) component, we raden aan deze versie te gebruiken.

## Voorbeeld

```js
import { VlStepsComponent } from '@domg-wc/components/next/steps';
```

```html
<vl-steps-next></vl-steps-next>
```
