# Icon - next

<VluxMetaData id="components-next-icon" />

Gebruik de `icon-next` component om een icoon af te beelden op een pagina.<br/>
Zie de [Alle Iconen](/docs/ontwerp-iconen-alle-iconen--documentatie) documentatie voor een overzicht van alle iconen die gebruikt kunnen worden.

Dit component is de volgende versie van het [vl-icon](/docs/elements-icon-icon--documentatie) element.


## Voorbeeld

```js
import { VlIconComponent } from '@domg-wc/components/next/icon';
```

```html
<vl-icon-next></vl-icon-next>
```
