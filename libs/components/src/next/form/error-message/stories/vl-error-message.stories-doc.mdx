# Error Message

Gebruik het `error-message-next` component om een error af te beelden voor een input veld.<br/>
Dit component is de volgende versie van het [vl-form-validation-message](?path=/docs/elements-form-message--form-validation-message-default) component, we raden aan deze versie te gebruiken.<br />

## Voorbeeld

```js
import { VlErrorMessageComponent } from '@domg-wc/components/next/form/error-message';
```

```html
<vl-error-message-next></vl-error-message-next>
```
