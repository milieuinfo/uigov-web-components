# Textarea

Gebruik de `textarea-next` component om een textarea veld toe te voegen aan een pagina.<br/>
Dit component is de volgende versie van het [vl-textarea](?path=/docs/components-textarea--textarea-default) component, we raden aan deze versie te gebruiken.<br />
In tegenstelling tot de vorige [vl-textarea](?path=/docs/components-textarea--textarea-default) bevat dit component geen rich-text modus, hier wordt op termijn een ander component voor voorzien.

## Voorbeeld

```js
import { VlTextareaComponent } from '@domg-wc/components/next/form/textarea';
```

```html
<vl-textarea-next></vl-textarea-next>
```
