# Radio Group - next

Gebruik de `radio-group-next` component om de gebruiker de mogelijkheid te geven om 1 keuze te selecteren in een lijst.<br/>
Wanneer mogelijk, selecteer geen `radio` van vooraf zodat de gebruiker een bewuste keuze kan maken.<br/>
De `radio-next` component is een onderdeel van de `radio-group-next` component.<br/>
Zie het [form demo](/docs/applicatief-voorbeelden-form-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Dit component is de volgende versie van het [vl-radio](/docs/components-radio--documentatie) component, we raden aan deze versie te gebruiken.

## Voorbeeld

```js
import { VlRadioComponent, VlRadioGroupComponent } from '@domg-wc/components/next/form/radio-group';
```

```html
<vl-radio-group-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
    <vl-radio-next></vl-radio-next>
</vl-radio-group-next>
```
