# Checkbox

Gebruik de `checkbox` component om de gebruiker de mogelijkheid te geven om 1 of meerdere dingen te selecteren in een lijst.<br/>
Dit component is de volgende versie van de [vl-checkbox](/docs/components-checkbox--documentatie) component, we raden aan deze versie te gebruiken.

## Voorbeeld

```js
import { VlCheckboxComponent } from '@domg-wc/components';
```

```html
<vl-checkbox-next></vl-checkbox-next>
```
