# Checkbox

Gebruik het `checkbox` component om de gebruiker de mogelijkheid te geven om 1 of meerdere dingen te selecteren in een lijst.<br/>

Dit component is de volgende versie van het [vl-checkbox](?path=/story/components-next-form-checkbox--checkbox-default) component, we raden aan deze versie te gebruiken.<br />
De oude [vl-checkbox](?path=/docs/components-checkbox--checkbox-default) versie is behouden voor backwards-compatibility, in v2.0.0 gaat dit verdwijnen.<br/><br/>


## Voorbeeld

```js
import { VlCheckboxComponent } from '@domg-wc/components';
```

```html
<vl-checkbox-next></vl-checkbox-next>
```
