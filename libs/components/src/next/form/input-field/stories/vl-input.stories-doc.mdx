# Input Field

Gebruik het `input-field-next` component om een input veld toe te voegen aan een pagina.<br/>
Dit component is de volgende versie van het [vl-input-field](?path=/docs/elements-input-field--input-field-default) component, we raden aan deze versie te gebruiken.<br />

## Voorbeeld

```js
import { VlInputFieldComponent } from '@domg-wc/components/next/form/input-field';
```

```html
<vl-input-field-next></vl-input-field-next>
```
