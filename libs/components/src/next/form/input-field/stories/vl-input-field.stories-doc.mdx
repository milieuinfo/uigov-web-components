# Input Field

Gebruik de `input-field-next` component om een input veld toe te voegen aan een pagina.<br/>
Dit component is de volgende versie van de [vl-input-field](/docs/elements-input-field--documentatie) component, we raden aan deze versie te gebruiken.<br />

## Voorbeeld

```js
import { VlInputFieldComponent } from '@domg-wc/components/next/form/input-field';
```

```html
<vl-input-field-next></vl-input-field-next>
```
