# Doormat - next

<VluxMetaData id="components-next-doormat" />

Gebruik de `doormat-next` component om een snel en duidelijk overzicht weer te geven van de informatie op je website.<br/>
Elke doormat krijgt zijn eigen titel, tekst, optionele afbeelding en linkt naar een pagina binnen je website.

Deze component is de volgende versie van de [vl-doormat](/docs/elements-doormat--documentatie) component.


## Voorbeeld

```js
import { VlDoormatComponent } from '@domg-wc/components/next/doormat';
```

```html
<vl-doormat-next></vl-doormat-next>
```
