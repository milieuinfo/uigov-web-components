# Textarea

<CustomDocMetaData id="components-textarea" />

Gebruik een `textarea` als invoervak in een formulier, waarin de gebruiker over meerdere regels tekst kan invoeren.


## Voorbeeld

```js
import { VlTextarea } from '@domg-wc/components';
```

```html
<textarea is="vl-textarea"></textarea>
```

<Canvas of={VlTextareaStories.TextareaDefault} />

## Variant

### Rich Text Editor

Wanneer je `data-vl-rich` op `true` zet wordt de textarea een WYSIWYG editor die achterliggend
[TinyMCE](https://www.tiny.cloud/) gebruikt.

<Canvas of={VlTextareaStories.TextareaRich} />
