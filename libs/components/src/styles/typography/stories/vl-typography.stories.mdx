# Typography - next

Gebruik de `vl-typography-next` styles om tekst en standaard HTML af te beelden op een pagina in de stijl van Digitaal Vlaanderen.

Deze CSS klasses vervangen het [vl-typography](/docs/components-typography--documentatie) component.

## Voorbeeld

```js
import { typographyStyles } from '@domg-wc/components/styles';
```

<Canvas>
    <Story name="vl-typography - default">
        {html`
            <style>
                ${typographyStyles.map((style) => unsafeCSS(style.cssText))}
            </style>
            <div class="vl-typography-next">
                <h1>heading 1</h2>
                <p><strong>strong-tag</strong></p>
                <p><b>b-tag</b></p>
                <p><em>em-tag</em></p>
                <p><i>i-tag</i></p>
                <p><s>s-tag</s></p>
                <p><ins>ins-tag</ins></p>
                <p><del>del-tag</del></p>
                <p>
                    <mark>mark-tag</mark>
                </p>
                <p><code>code-tag</code></p>
                <pre>pre-tag</pre>
                <blockquote>Lorem ipsum dolor sit amet.</blockquote>
                <ol>
                    <li>item 1</li>
                    <li>item 2</li>
                    <li>item 3</li>
                </ol>
                <a href="#">link<a>
            </div>
    `}
    </Story>
</Canvas>
