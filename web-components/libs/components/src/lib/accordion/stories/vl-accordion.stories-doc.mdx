# Accordion

De accordion component kan gebruikt worden om informatie te tonen of te verbergen aan de hand van een toggle.

## Voorbeeld

```js
import { VlAccordionComponent } from '@domg-wc/components';
```

```html
<vl-accordion data-vl-toggle-text="Lees meer over de onderwijsdoelstelling">
    <span>
        Onderwijs helpt jonge mensen en volwassenen om zichzelf te ontwikkelen en hun weg te vinden in onze samenleving.
        Het hoger onderwijs speelt daarnaast een belangrijke rol in innovatie dankzij het belang van wetenschappelijk
        onderzoek.
    </span>
</vl-accordion>
```
