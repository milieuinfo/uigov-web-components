## Planning

## v1.40.x

Dit is een release met 'next' impact.

- de herwerking met het 'token' en het 'common-legacy' artifact
- de zaken waar al aan gewerkt is releasen
  * introduction - beschikbaar in de branch van ticket [UIG-2923]
  * text - beschikbaar in de branch van ticket [UIG-2933]
- de pure css next-componenten
  * action-group -> 'css'-component: 'vl-group' (*)
  * link-list -> 'css'-component: 'vl-group' (*)
  * icon-wrapper -> 'css'-component: 'vl-group' (*)
  * form-grid -> 'css'-component: 'vl-grid' (**)
  * grid -> 'css'-component: 'vl-grid' (**)
- de te verwijderen componenten bekijken
  * image -> kan (vermoedelijk) weg, als het iets nuttig doet in de globale css dat voorzien op `<img>`
  * body -> kan (vermoedelijk) weg, als het iets nuttig doet in de globale css dat voorzien op `<body>`
