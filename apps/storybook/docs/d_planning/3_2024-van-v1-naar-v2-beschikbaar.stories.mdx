## Aanpak

Op deze pagina vind je een overzicht van de web componenten die geïmpacteerd zijn door de migratie van v1 naar v2
en die reeds beschikbaar zijn. Voor de resterende componenten (de geïmpacterede die we nog gaan implementeren) is er
de pagina [[2024] van v1 naar v2 - planning](/docs/planning-2024-van-v1-naar-v2-planning--documentatie).

Hieronder vind je alfabetisch, per component, een tabel met daarin een overzicht van wat er in v1 en v2
beschikbaar is (zal zijn), je kan doorklikken op de naam om naar de documentatie van die component te gaan.

- elke component is ontdubbeld, de legacy variant staat in een donkere rij, de next variant in de erop volgende
  witte rij
- voor de next variant staat gespecifieerd sinds welke release hij beschikbaar is
- typisch hebben de legacy en de next variant dezelfde naam, maar niet altijd
- momenteel zijn er enkel v1 releases, wat is er zal gebeuren in v2 is informatief; er is nog geen v2 beschikbaar
