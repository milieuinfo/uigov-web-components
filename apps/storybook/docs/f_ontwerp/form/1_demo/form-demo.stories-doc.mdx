## Native elementen

Gebruik voor de volgende elementen native HtmlElements met css classes:
- Form: native HTMLFormElement met class `vl-form`
- Button: native HtmlButtonElement met class `vl-button`
- Action Group: native HtmlDivElement met class `vl-action-group`

## Grid

Maak voor het opbouwen van de grid structuur gebruik van de [vl-form-grid classes](https://overheid.vlaanderen.be/webuniversum/v3/documentation/form-grid) van Digitaal Vlaanderen.<br/>
Plaats deze op native HtmlDivElements.

## Form data

Gebruik de [parseFormData](/docs/ontwerp-form-form-data--documentatie) helper functie om de form data te verzamelen.<br/>
Indien je meer controle nodig hebt kan je zelf de [FormData](https://developer.mozilla.org/en-US/docs/Web/API/FormData) API gebruiken.

## ‚ùóLet op

Indien je een form control wrapped in een custom component, zorg er voor dat dit custom component geen Shadow DOM heeft.<br/>
Als dit custom component een Shadow DOM heeft, zal het native HtmlFormElement de form control niet meer vinden.
