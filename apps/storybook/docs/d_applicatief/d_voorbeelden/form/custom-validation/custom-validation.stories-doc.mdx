# Custom Validatie

Om gebruik te maken van custom validatie moet je een nieuw component maken dat overerft van één van onze form controls.<br/>
Aan dit component kan je je custom validator toevoegen. Via de `customError` ValidityState key kan je een error message linken aan je custom validator.

## Extra informatie

Het is nodig om een custom component te maken voor elke custom validator die je wil toevoegen:<br/>
- De validators worden toegevoegd op de form controls dmv een statiche array. Dit betekent dat als je een custom validator toevoegt aan een form control, deze validator zal toegevoegd worden aan alle form controls van dit type.<br/>
- De form controls maken gebruik van de ValidityState API waardoor er maximum 1 custom validator toegevoegd kan worden aan een form control.<br/>
Bij het toevoegen van een 2de custom validator zal de eerste custom validator overschreven worden.<br/>
