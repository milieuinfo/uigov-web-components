# GitHub Pages

## Beschrijving

Documentatie wordt geversioneerd gepubliceerd op https://milieuinfo.github.io/uigov-pages, de structuur van die repo
staat hier beschreven.

## Structuur

```
uigov-pages/
├─ build-apps/ [de sub-folders zijn de branches]
│  ├─ develop/
│  ├─ develop-apo/
│  │  ├─ xxx/
│  ├─ release/ [een sub-folder per versie - uit de root package.json van web-components]
│  │  ├─ 1.0.0/
│  │  │  ├─ alliance/
│  │  │  ├─ playground/
│  │  │  ├─ storybook/
├─ jekyll/
│  -> basis jekyll structuur
│  -> de build folders zijn hierin opgenomen via een symbolic link (zie de .github workflow)
│  -> er is gestart op basis van https://jekyllthemes.io/theme/doxy-jekyll-theme
├─ processors/
│  -> NX repo met een 'version-processing' plugin om een .yml file op te bouwen die de versie structuur
│     bevat en gebruikt wordt door jekyll
├─ _site/
│  -> de gegenereerde website die gepubliceerd wordt onder https://milieuinfo.github.io/uigov-pages/
```
